# 🔐 Phishing Awareness Training System

A complete web-based cybersecurity training platform designed to educate users about phishing attacks and how to stay safe online.

---

## 🚀 Project Overview

This project simulates a real-world cybersecurity awareness system where users:

* Enter basic details (Login Simulation)
* Learn about phishing attacks
* Take an interactive quiz
* Receive a certificate upon completion

---

## 🎯 Features

* 🔐 Secure Login Simulation (Name & Mobile Input)
* 🧠 Phishing Awareness Content
* ❓ Interactive Quiz (10 Questions)
* ⚡ Auto Next Question System
* 🏆 Certificate Generation with User Name & Date
* 🎨 Cybersecurity UI (Hacker Style Design)

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

---

## 📚 What You Will Learn

* What is phishing?
* Types of phishing attacks
* How attackers trick users
* How to identify fake links
* Cybersecurity best practices

---

## ⚙️ How It Works

1. User opens the website
2. Login page appears (Name & Mobile)
3. Redirect to awareness page
4. User starts quiz
5. After completion → certificate is generated
